# GeoPulse Frontend (Preact + Vite)

This is the migrated frontend running on Preact compatibility mode with Vite.

## Scripts

- `npm run dev` — start dev server
- `npm run build` — build production bundle
- `npm run preview` — preview production bundle locally

## Environment

Copy `.env.example` to `.env` and set:

- `REACT_APP_BACKEND_URL` (default backend URL)
- `ENABLE_HEALTH_CHECK` (optional plugin toggle)
